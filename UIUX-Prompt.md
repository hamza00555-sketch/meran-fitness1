# برومبت توثيق UI/UX — تطبيق راتبي 💼

---

## نظرة عامة على التطبيق

**راتبي** هو تطبيق ويب تقدمي (PWA) لإدارة الراتب الشهري باللغة العربية، يعمل بالكامل offline. يساعد المستخدم على توزيع راتبه الشهري بين الالتزامات والأهداف، وتتبع مدفوعاته عبر حساباته البنكية المختلفة.

---

## نظام التصميم

### الألوان
| الاستخدام | اللون |
|-----------|-------|
| الخلفية الرئيسية | `#0D0B2A` (كحلي داكن جداً) |
| الكاردات / السطح | `#151235` — `#1A1650` |
| اللون الأساسي (Primary) | `#6C63FF` (بنفسجي) |
| التأكيد الثانوي | `#43B89C` (تركواز / أخضر) |
| الالتزامات / تحذير | `#FF6584` (وردي / أحمر) |
| النص الأساسي | `#FFFFFF` |
| النص الثانوي | `#9B99C8` (رمادي بنفسجي) |
| الحدود | `#2A2660` |

### الخطوط
- الخط الأساسي: **Mestika** (عربي مخصص) + Cairo كـ fallback
- اتجاه النص: RTL (من اليمين لليسار)
- الأوزان المستخدمة: 400 / 700 / 900

### الأنماط المتكررة
- **زوايا مدورة**: 16–24px للكاردات، 12px للأزرار الصغيرة
- **ظلال**: `box-shadow: 0 4px 24px rgba(0,0,0,.4)`
- **تأثير Glassmorphism خفيف** على بعض العناصر
- **FAB Button** (زر عائم دائري بنفسجي ＋) في كل صفحة للإضافة

---

## هيكل التنقل

شريط تنقل سفلي ثابت (Bottom Navigation) يحتوي على 5 تبويبات:

```
[ الرئيسية 🏠 ] [ التزاماتي 📋 ] [ بنوكي 🏦 ] [ أهدافي 🎯 ] [ الإعدادات ⚙️ ]
```

---

## الشاشات التفصيلية

---

### 1. شاشة الإعداد الأولي (Onboarding) — 3 خطوات

**الخطوة 1 من 3: الراتب الشهري**
- شعار التطبيق (أيقونة محفظة + نص "راتبي") في الزاوية العلوية اليمنى
- رسمة توضيحية لشخص جالس (Illustration)
- عنوان ضخم: "تحكم في مالك، وحقق أهدافك."
- نص فرعي: "ابدأ بإدخال راتبك الشهري"
- مؤشر التقدم: "الخطوة ١ من ٣" (بالأحمر/البنفسجي)
- حقل إدخال: الراتب الشهري (ريال) — placeholder: "مثال: 10000"
- منتقي يوم الراتب: شبكة أزرار 7 أعمدة × 5 صفوف (الأيام 1–31)
  - اليوم المحدد: خلفية بنفسجية + نص أبيض
  - باقي الأيام: خلفية كارد داكن

**الخطوة 2 من 3: إضافة الالتزامات**
- قائمة الالتزامات الموجودة (فارغة في البداية)
- زر "إضافة التزام" يفتح Bottom Sheet

**الخطوة 3 من 3: إضافة الأهداف**
- مشابه للخطوة 2 لكن للأهداف

---

### 2. الرئيسية (Dashboard)

#### الجزء العلوي
- **شارة الراتب** (يسار): كارد تركواز — نص "الراتب" صغير + المبلغ كبير (مثال: 12,000 ريال)
- **زر إخفاء الأرقام** 👁 (يمين الشارة): تحويل كل الأرقام إلى ••••
- **التاريخ الهجري** (يمين أعلى): مثال "ذو الحجة ١٤٤٧ هـ"
- **اسم التطبيق** "راتبي 💼" (يمين وسط)

#### مخطط التوزيع (Donut Chart)
- **رسم دائري donut** بثلاثة ألوان:
  - 🔴 أحمر: الالتزامات (مثال: 4,815)
  - 🟣 بنفسجي فاتح: الأهداف (مثال: 3,500)
  - 🟢 تركواز: المتبقي (مثال: 3,685)
- في المنتصف: نص "متبقي" + المبلغ بخط كبير + "ريال"
- جانب المخطط (يمين): أسطورة الألوان مع المبالغ

#### بطاقتا الإحصاء
- **2 كارد** في صف أفقي:
  - 🎯 أهداف الشهر: المبلغ بالبنفسجي
  - 📋 التزامات الشهر: المبلغ بالأحمر

#### الدخل الإضافي 💰
- عنوان "الدخل الإضافي 💰" + زر "+ إضافة" (يسار)
- قائمة بطاقات الدخل الإضافي، كل بطاقة تحتوي:
  - زر ✕ لحذف الدخل (يسار)
  - تاريخ الدخل (مثال: "دخل ٢٠ ذو الحجة") + أيقونة 💰
  - المبلغ بخط ضخم (مثال: 3,000 ريال)
  - توزيع الدخل: 🏦 1,500 (ديون) · 🏷 1,000 (أهداف) · 🎁 500 (شخصي)

#### توزيع التحويلات 🏦
- عنوان "توزيع التحويلات 🏦" + رابط "عرض البنوك"
- كارد لكل بنك يحتوي:
  - اسم البنك + أيقونة البنك (ملونة بلون البنك في الحدود اليمنى)
  - **إجمالي التحويل** (أحمر) + "ريال"
  - لكل حساب: اسم الحساب + قائمة بالالتزامات والأهداف المعينة
    - 📋 الالتزام + المبلغ (أحمر)
    - 🎯 الهدف + المساهمة الشهرية (بنفسجي)

#### تقدم الأهداف 🎯
- عنوان "تقدم الأهداف 🎯" + رابط "عرض الكل"
- كارد مبسط لكل هدف:
  - اسم الهدف + أيقونة الفئة
  - المبلغ المحفوظ / المستهدف
  - شريط تقدم ملون (gradient من بنفسجي إلى تركواز)
  - نسبة التقدم على اليسار

#### الالتزامات القادمة ⏰
- كارد لكل التزام في الـ 7 أيام القادمة
- باج ملون يوضح:
  - 🔴 "غداً" / "بعد X أيام" (قريب)
  - 🟡 "بعد 5 أيام"

### 3. التزاماتي (Commitments)

#### الرأس
- عنوان "التزاماتي" (يمين)
- عداد مدفوع: "2 / 7 مدفوع" (يمين، أسفل العنوان)
- إجمالي الالتزامات: "4,815 ريال / شهر" (يسار، أحمر)

#### فلاتر التصفية (3 أزرار Pill-style)
- **الكل** (محدد: بنفسجي) · **قادم** · **مدفوع**

#### كارد الالتزام
كل التزام يظهر في بطاقة مستقلة تحتوي:
- **اسم الالتزام** (كبير، يمين)
- **الحالة badge** (يمين، تحت الاسم):
  - ✓ مدفوع (أخضر، مع خط يتوسط اسم الالتزام)
  - بعد X أيام (برتقالي/أحمر)
  - يوم X (رمادي)
- **البنك والحساب** 🏦 مثال "الحساب الجاري" (في المنتصف)
- **المبلغ** (يسار، كبير، أحمر)
- **زر التأكيد** ✓ (صغير، يؤدي toggle للحالة):
  - غير مدفوع: خلفية رمادية داكنة
  - مدفوع: خلفية تركواز
- **زر المزيد** ••• (الزاوية اليسارية العلوية): يفتح قائمة للتعديل/الحذف
- **أيقونة المرفق** 📎 (صغيرة، تحت زر التأكيد)

**FAB Button**: زر "+" بنفسجي دائري في أسفل اليمين — يفتح Bottom Sheet لإضافة التزام جديد

#### نموذج إضافة التزام (Bottom Sheet)
حقول:
- اسم الالتزام
- المبلغ (ريال)
- فئة: أيقونة ملونة (سكن، سيارة، صحة، اشتراكات، طعام، تعليم، ترفيه، أخرى)
- يوم الدفع: منتقي رقمي 1–31
- بنك وحساب: قوائم اختيار
- تعيين للدخل الإضافي: checkbox

---

### 4. بنوكي (Banks)

#### الرأس
- عنوان "بنوكي" (يمين)
- نص توضيحي: "وجهات التحويل لالتزاماتك وأهدافك · 3 بنك"

#### كارد البنك
- **حدود يمنى ملونة** بلون البنك المخصص
- أيقونة البنك (emoji داخل دائرة ملونة) + اسم البنك + عدد الحسابات
- زر 📎 (يسار): تعديل البنك
- لكل حساب: **كارد داخلي** يحتوي:
  - اسم الحساب (يمين، أبيض)
  - قائمة الالتزامات المعينة: 📋 اسم الالتزام — المبلغ (أحمر)
  - قائمة الأهداف المعينة: 🎯 اسم الهدف — X ريال/شهر (بنفسجي)
  - إذا لا يوجد تعيينات: نص "لا يوجد تعيينات بعد" (رمادي)

**FAB Button**: "+ بنك جديد"

---

### 5. أهدافي (Goals)

#### الرأس
- عنوان "أهدافي" (يمين)
- عداد: "3 هدف جارٍ · 1 مكتمل"
- زر "🧮 الحاسبة" (يسار): يفتح حاسبة الادخار

#### كارد الهدف (Active)
- **أيقونة الفئة** ملونة (يمين، كبيرة): سفر، تقنية، ادخار، سيارة...
- **اسم الهدف** (كبير، يمين)
- **التقدم المالي**: "6,200 / 15,000 ريال"
- **badge البنك**: 🏦 اسم الحساب (ملون بلون البنك)
- **نسبة التقدم**: عدد كبير على اليسار "٪41"
- **شريط التقدم**: gradient بنفسجي–تركواز، متوهج
- **معلومات أسفل الشريط**: "1,200 ريال / شهر · 1 شهر متبقي"
- **أزرار الإجراءات** (صف سفلي):
  - "+ إضافة" (تركواز): إضافة مبلغ للهدف
  - 📎 (رمادي): تعديل الهدف

#### كارد الهدف (مكتمل ✓)
- نفس التصميم لكن بـ **opacity 70%**
- شارة "✓ مكتمل" تحت الاسم (خضراء)
- شريط التقدم 100% (تركواز كامل)
- لا يوجد أزرار إجراءات

**FAB Button**: "+ هدف جديد"

### 6. الإعدادات (Settings)

#### قسم إعدادات الراتب 💰
- عنوان القسم (بنفسجي)
- حقل "الراتب الشهري (ريال)": input رقمي
- منتقي "يوم نزول الراتب": شبكة أزرار 1–31 (نفس نمط الـ Onboarding)
  - اليوم المحدد: بنفسجي
- زر "حفظ الإعدادات" (أزرق بنفسجي كامل العرض)

#### قسم استيراد البيانات 🤖
- عنوان "استيراد البيانات 🤖"
- تعليمات استخدام الـ AI (4 خطوات مرقمة):
  1. انسخ "برومبت الذكاء الاصطناعي" أدناه
  2. أرسله لـ ChatGPT وأخبره بمعلوماتك
  3. الصق الرد هنا مكان القالب
  4. اضغط استيراد
- زر "نسخ برومبت الذكاء الاصطناعي 🤖"
- صندوق نصي للقالب المُنتَج مع زر "نسخ 📋"
- زر "⬇️ استيراد البيانات" (بنفسجي كامل العرض)

#### قسم يوم الراتب 📅
- زر "فتح شاشة توزيع الراتب مجدداً لهذا الشهر"

#### قسم الأمان والخصوصية 🔒
- toggle "تفعيل قفل التطبيق بالرمز السري"
- عند التفعيل: يظهر منتقي الـ PIN (4 أرقام)
- toggle "بصمة الوجه / الإصبع"
- معلومات عن التخزين المحلي

#### قسم تحديث التطبيق 🔄
- معلومات الإصدار الحالي
- زر "تحديث التطبيق"

### 7. شاشة القفل (Lock Screen)

- خلفية: `var(--bg)` الداكنة
- **أيقونة التطبيق** (88×88px، زوايا مدورة، ظل ضخم)
- **اسم التطبيق** "راتبي" (كبير، بولد)
- **نص** "أدخل رمز الدخول"
- **4 نقاط PIN** (فارغة: رمادية / مملوءة: بنفسجية متوهجة)
- **لوحة أرقام** (Numpad):
  - شبكة 3×4 — direction: LTR (أرقام بترتيب صحيح)
  - الصف 1: 1 · 2 · 3
  - الصف 2: 4 · 5 · 6
  - الصف 3: 7 · 8 · 9
  - الصف 4: [بصمة الوجه 👁] · 0 · [⌫ حذف]
  - كل زر: 70px ارتفاع، border-radius 16px، ظل
- **زر "الدخول ببصمة الوجه / الإصبع"** (نص بنفسجي أسفل اللوحة)
- عند الخطأ: **اهتزاز shake animation** للنقاط + رسالة خطأ حمراء

---

### 8. شاشة توزيع الراتب (Salary Day)

تظهر تلقائياً في يوم الراتب إذا لم يُوزَّع الراتب بعد:
- عنوان: "يوم الراتب 🎉"
- المبلغ المستلم
- توزيع تلقائي على الالتزامات والأهداف
- ملخص "سيُحوَّل إلى:"
  - قائمة البنوك مع المبالغ
- زر "تأكيد التوزيع" (بنفسجي)

---

### 9. نافذة الدخل الإضافي (ExtraIncomeSheet)

**Bottom Sheet** يُفتح من الـ Dashboard:
- عنوان "إضافة دخل إضافي"
- حقل المبلغ
- حقل المصدر (مثال: فريلانس، بيع، مكافأة)
- قسم التوزيع: شريط تفاعلي لتوزيع المبلغ على:
  - 🏦 سداد الديون
  - 🎯 الأهداف
  - 🎁 الاستخدام الشخصي
- زر "حفظ الدخل" (بنفسجي كامل العرض)

## خلاصة الـ UX Flows الرئيسية

```
أول تشغيل:
  Onboarding (3 خطوات) → Dashboard

كل شهر (يوم الراتب):
  SalaryDay Screen → توزيع تلقائي → Dashboard

تسجيل التزام مدفوع:
  Commitments → ضغط ✓ → badge يتحول لـ "مدفوع"

إضافة دخل إضافي:
  Dashboard → "+ إضافة" → ExtraIncomeSheet → حفظ → يظهر في الـ Dashboard

تتبع الهدف:
  Goals → زر "+إضافة" → إدخال المبلغ → شريط التقدم يتحدث

فتح التطبيق مع قفل:
  LockScreen → PIN أو بصمة → Dashboard
```

## ملاحظات تقنية للمطور

- **تقنية**: React 18 + Vite + PWA (Service Worker)
- **التخزين**: IndexedDB (idb library) — كل البيانات محلية 100%
- **RTL**: التطبيق بالكامل RTL، الشبكات الرقمية (numpad) تستخدم `direction: ltr`
- **Safe Area**: `env(safe-area-inset-bottom)` مضاف على كل الشاشات
- **أنيميشن**: `@keyframes slideUp` للـ Bottom Sheets، `fadeIn` لتحميل الكاردات
- **الخط**: `Mestika` خط عربي مخصص يُحمَّل من `/assets/fonts/`
- **الأيقونة**: المحفظة الخضراء الداكنة كـ PWA icon
